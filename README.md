# CSADPRG MP
 
To run the main program, do:
```
go run main.go
```
To run the tests, do:
```
cd calc
go test
```
To get code coverage, do:
```
cd calc
go test -coverprofile=coverage.out
go tool cover -html=coverage.out
```
To format code, do:
```
go fmt
cd calc
go fmt
```
